Sanic-RESTful-Api
=============

.. module:: sanic_restful_api

**Sanic-RESTful-Api** is an extension for Flask that adds support for quickly
building REST APIs.  It is a lightweight abstraction that works with your
existing ORM/libraries.  Sanic-RESTful-Api encourages best practices with minimal
setup. If you are familiar with Flask, Sanic-RESTful-Api should be easy to pick up.
(such as `flask_restful <https://github.com/flask-restful/flask-restful>`_)
