.. _installation:

Installation
============

.. currentmodule:: sanic_restful_api

Install Sanic-RESTful-Api with ``pip`` ::

    pip install sanic-restful-api


The development version can be downloaded from `its page at GitHub
<https://github.com/linzhiming0826/sanic-restful>`_. ::

    git clone https://github.com/linzhiming0826/sanic-restful.git
    cd sanic-restful
    python setup.py develop


Sanic-RESTful-Api has the following dependencies (which will be automatically
installed if you use ``pip``):


* `Sanic <https://sanicframework.org/>`_ version 18.12.0 or greater

Sanic-RESTful-Api requires Python version 3.6 or greater
